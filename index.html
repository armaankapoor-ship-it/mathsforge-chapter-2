01Basics

Why inverse trig needs branches

sin x, cos x, and tan x are not one-one on their full domains, so we restrict them to principal branches.

  • CBSE writing: Mention one-one restriction before defining inverse.
  • JEE focus: Branch controls sign and final angle.
  • Common trap: Inverse trig gives principal value, not all solutions.
02Principal values

sin^-1 x

sin^-1 x has domain [-1,1] and range [-pi/2, pi/2].

  • CBSE writing: State domain and range clearly.
  • JEE focus: Output angle has cosine non-negative.
  • Common trap: sin^-1(sin theta) is not always theta.
03Principal values

cos^-1 x

cos^-1 x has domain [-1,1] and range [0, pi].

  • CBSE writing: Remember output is never negative.
  • JEE focus: Use cos^-1(-x)=pi-cos^-1 x.
  • Common trap: Do not use [-pi/2,pi/2] for cos inverse.
04Principal values

tan^-1 x

tan^-1 x has domain R and range (-pi/2, pi/2).

  • CBSE writing: Use open interval for range.
  • JEE focus: Addition formula needs case handling.
  • Common trap: tan^-1 x + tan^-1 y is not always tan^-1((x+y)/(1-xy)).
05Principal values

cot^-1 x

cot^-1 x has domain R and range (0, pi).

  • CBSE writing: Use positive principal range.
  • JEE focus: cot^-1 x = pi/2 - tan^-1 x under NCERT convention.
  • Common trap: Different books may use different cot inverse ranges; follow NCERT/CBSE.
06Principal values

sec^-1 x and cosec^-1 x

Both require |x| >= 1. Their ranges exclude angles where sec or cosec is undefined.

  • CBSE writing: Write domain R - (-1,1).
  • JEE focus: Use sec^-1 x = cos^-1(1/x), cosec^-1 x = sin^-1(1/x) with range care.
  • Common trap: x cannot lie between -1 and 1.
07Graphs

Graph of inverse functions

Inverse function graph is reflection of the restricted original graph in y=x.

  • CBSE writing: Draw branch and reflection neatly.
  • JEE focus: Use graph to judge range and monotonicity.
  • Common trap: Reflect only the restricted branch, not entire trig graph.
08Identities

Complementary identities

sin^-1 x + cos^-1 x = pi/2 and tan^-1 x + cot^-1 x = pi/2.

  • CBSE writing: Mention x in domain.
  • JEE focus: These save time in simplification.
  • Common trap: Do not apply outside the common domain.
09Identities

Negative argument identities

sin^-1(-x)=-sin^-1 x and tan^-1(-x)=-tan^-1 x; cos^-1(-x)=pi-cos^-1 x.

  • CBSE writing: Separate odd and non-odd inverse functions.
  • JEE focus: Use sign to bring arguments into principal range.
  • Common trap: cos^-1 is not odd.
10Identities

tan inverse addition

tan^-1 x + tan^-1 y depends on xy and signs because principal value range is limited.

  • CBSE writing: Use formula with stated condition xy<1 for simple form.
  • JEE focus: When xy>1, add or subtract pi according to signs.
  • Common trap: Ignoring cases gives wrong quadrant.
11Equations

Solving inverse trig equations

Convert the inverse expression to a principal angle, then apply domain and range restrictions.

  • CBSE writing: Write domain restrictions before solving.
  • JEE focus: Check extraneous answers from transformations.
  • Common trap: Squaring or applying trig can add invalid roots.
12Applications

Simplification strategy

Name the inverse angle, draw a triangle, then compute required trig value.

  • CBSE writing: Triangle method is clean for expressions like sin(tan^-1 x).
  • JEE focus: Triangle method avoids memorizing too many identities.
  • Common trap: Choose side signs according to principal range.

Domain

sin inverse

sin^-1 x: domain [-1,1], range [-pi/2, pi/2]

Returns the principal angle whose sine is x.

  • Example: sin^-1(1/2)=pi/6.
  • Trap: Output cannot be 5pi/6.
Domain

cos inverse

cos^-1 x: domain [-1,1], range [0, pi]

Returns the principal angle whose cosine is x.

  • Example: cos^-1(-1/2)=2pi/3.
  • Trap: Output is not negative.
Domain

tan inverse

tan^-1 x: domain R, range (-pi/2, pi/2)

Returns principal angle with tangent x.

  • Example: tan^-1(1)=pi/4.
  • Trap: Endpoints are excluded.
Domain

cot inverse

cot^-1 x: domain R, range (0, pi)

Returns principal angle with cotangent x.

  • Example: cot^-1(1)=pi/4.
  • Trap: Follow NCERT range convention.
Domain

sec inverse

sec^-1 x: domain (-infinity,-1] U [1,infinity)

Defined only when reciprocal lies in [-1,1].

  • Example: sec^-1(2)=cos^-1(1/2)=pi/3.
  • Trap: sec^-1(1/2) is undefined in real numbers.
Identity

Complementary identity

sin^-1 x + cos^-1 x = pi/2

Sine and cosine principal angles are complementary.

  • Example: sin^-1(1/2)+cos^-1(1/2)=pi/2.
  • Trap: x must be in [-1,1].
Identity

Tan-cot identity

tan^-1 x + cot^-1 x = pi/2

Principal tan and cot angles are complementary.

  • Example: tan^-1(sqrt3)+cot^-1(sqrt3)=pi/2.
  • Trap: Range convention matters.
Identity

Negative argument

sin^-1(-x)=-sin^-1 x, tan^-1(-x)=-tan^-1 x

sin inverse and tan inverse are odd.

  • Example: tan^-1(-1)=-pi/4.
  • Trap: cos inverse is not odd.
Identity

Cos negative argument

cos^-1(-x)=pi-cos^-1 x

Cos inverse range is [0,pi].

  • Example: cos^-1(-1/2)=pi-pi/3=2pi/3.
  • Trap: Do not write -cos^-1 x.
Formula

Tan inverse addition

tan^-1 x + tan^-1 y = tan^-1((x+y)/(1-xy)) when xy<1

Simple addition case for arctan.

  • Example: tan^-1(1/2)+tan^-1(1/3)=tan^-1(1).
  • Trap: If xy>1, handle pi adjustment.
Formula

Tan inverse subtraction

tan^-1 x - tan^-1 y = tan^-1((x-y)/(1+xy))

Difference formula within principal range.

  • Example: tan^-1 2 - tan^-1 1 = tan^-1(1/3).
  • Trap: Check denominator sign and final range.
Triangle

Triangle method

If theta=tan^-1 x, use opposite=x, adjacent=1, hypotenuse=sqrt(1+x^2)

Converts inverse trig expressions into ordinary trig values.

  • Example: sin(tan^-1 x)=x/sqrt(1+x^2).
  • Trap: Use correct quadrant from principal range.

Derivation bank

Board-ready derivations and proofs.

Each proof is broken into steps so students can reproduce it in exams.

Derive sin^-1 x + cos^-1 x = pi/2 pi/2
  1. Let theta=sin^-1 x, so theta in [-pi/2,pi/2] and sin theta=x.
  2. Then cos(pi/2-theta)=sin theta=x.
  3. Since pi/2-theta lies in [0,pi], it is the principal value for cos^-1 x.
  4. Thus cos^-1 x=pi/2-theta and the sum is pi/2.
  • Shortcut: Use complementary angles.
  • Warning: The range check is essential.
Derive cos^-1(-x)=pi-cos^-1 x pi - cos^-1 x
  1. Let theta=cos^-1 x, so theta in [0,pi].
  2. cos(pi-theta)=-cos theta=-x.
  3. pi-theta also lies in [0,pi].
  4. Therefore cos^-1(-x)=pi-theta.
  • Shortcut: Cos changes sign under pi-theta.
  • Warning: Do not treat cos inverse as odd.
Derive tan inverse addition formula tan^-1((x+y)/(1-xy))
  1. Let A=tan^-1 x and B=tan^-1 y.
  2. Then tan A=x and tan B=y.
  3. tan(A+B)=(x+y)/(1-xy).
  4. If A+B lies in the principal range, A+B equals tan^-1 of that expression.
  • Shortcut: Name angles first.
  • Warning: Case adjustment is needed when A+B leaves (-pi/2,pi/2).
Derive sin(tan^-1 x) x/sqrt(1+x^2)
  1. Let theta=tan^-1 x.
  2. In a right triangle, take opposite=x and adjacent=1.
  3. Hypotenuse is sqrt(1+x^2).
  4. Therefore sin theta=x/sqrt(1+x^2).
  • Shortcut: Triangle method beats memorization.
  • Warning: Use sign carefully for negative x.
Graph inverse by reflection reflect in y=x
  1. Restrict the trig function to a one-one branch.
  2. Take its graph on that branch.
  3. Swap x and y coordinates.
  4. This reflects the graph in y=x and gives the inverse graph.
  • Shortcut: Inverse graph is coordinate swap.
  • Warning: Reflect only the restricted branch.

Practice zone

Original practice questions with solutions.

Use the filter to isolate CBSE, JEE, proof, graph, or trap-based questions.

CBSEEasy

1. Find sin^-1(1/2).

Answer: pi/6

The principal angle in [-pi/2,pi/2] with sine 1/2 is pi/6.

Trap: 5pi/6 is not in the range of sin^-1.

CBSEEasy

2. Find cos^-1(-1/2).

Answer: 2pi/3

cos(2pi/3)=-1/2 and 2pi/3 lies in [0,pi].

Trap: Do not give -pi/3.

CBSEEasy

3. State domain of sec^-1 x.

Answer: (-infinity,-1] U [1,infinity)

sec theta=x means cos theta=1/x, so |1/x|<=1 and |x|>=1.

Trap: Values between -1 and 1 are not allowed.

IdentityMedium

4. Simplify sin^-1 x + cos^-1 x.

Answer: pi/2

For x in [-1,1], principal sine and cosine angles are complementary.

Trap: Mention domain.

IdentityMedium

5. Simplify cos^-1(-x).

Answer: pi - cos^-1 x

If theta=cos^-1 x, then cos(pi-theta)=-x.

Trap: cos inverse is not odd.

JEEMedium

6. Find tan^-1(1/2)+tan^-1(1/3).

Answer: pi/4

Use formula: (1/2+1/3)/(1-1/6)=1, so result is tan^-1 1=pi/4.

Trap: Check xy<1.

TriangleMedium

7. Find sin(tan^-1 x).

Answer: x/sqrt(1+x^2)

Let theta=tan^-1 x. Use opposite=x, adjacent=1, hypotenuse=sqrt(1+x^2).

Trap: Do not write x directly.

GraphMedium

8. What is the range of tan^-1 x?

Answer: (-pi/2, pi/2)

The principal branch of tan is restricted to (-pi/2,pi/2).

Trap: Endpoints are not included.

TrapHard

9. Is sin^-1(sin 2pi/3)=2pi/3?

Answer: No, it equals pi/3.

sin 2pi/3=sqrt3/2. The principal angle for sin inverse is pi/3.

Trap: sin^-1(sin theta) returns theta only when theta is in principal range.

JEEHard

10. Find tan^-1 2 + tan^-1 3.

Answer: pi + tan^-1(-5)

xy=6>1 and both positive, so the sum is in quadrant II. Formula gives tan value -5, so principal adjustment gives pi + tan^-1(-5).

Trap: Using tan^-1(-5) alone gives a negative angle.

CBSEMedium

11. Find cot^-1(-1).

Answer: 3pi/4

cot theta=-1 and theta in (0,pi), so theta=3pi/4.

Trap: Do not use -pi/4 under NCERT range.

ProofHard

12. Prove tan^-1 x + cot^-1 x = pi/2.

Answer: Use cot^-1 x = pi/2 - tan^-1 x.

Let theta=tan^-1 x. Then cot(pi/2-theta)=tan theta=x, and pi/2-theta lies in cot inverse range.

Trap: Range convention must be stated.

Must-know formulas

  • sin inverse: sin^-1 x: domain [-1,1], range [-pi/2, pi/2]
  • cos inverse: cos^-1 x: domain [-1,1], range [0, pi]
  • tan inverse: tan^-1 x: domain R, range (-pi/2, pi/2)
  • cot inverse: cot^-1 x: domain R, range (0, pi)
  • sec inverse: sec^-1 x: domain (-infinity,-1] U [1,infinity)
  • Complementary identity: sin^-1 x + cos^-1 x = pi/2
  • Tan-cot identity: tan^-1 x + cot^-1 x = pi/2
  • Negative argument: sin^-1(-x)=-sin^-1 x, tan^-1(-x)=-tan^-1 x
  • Cos negative argument: cos^-1(-x)=pi-cos^-1 x
  • Tan inverse addition: tan^-1 x + tan^-1 y = tan^-1((x+y)/(1-xy)) when xy<1

Must-avoid traps

  • Inverse trig returns principal value only.
  • cos^-1(-x)=pi-cos^-1 x, not -cos^-1 x.
  • tan^-1 addition formula needs case checking.
  • sec^-1 and cosec^-1 require |x|>=1.
  • cot^-1 range must follow NCERT convention.
  • Always check domain before simplifying.
